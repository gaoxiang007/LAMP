## How to install Mysql on Linux
1) Download the Mysql RPM
http://dev.mysql.com/downloads/mysql/

Download
Mysql Client
Mysql server
Mysql Devel

2) Install the RPM
rmp -ivh xxxxx

3) Start Mysql server

4) Change password

5) service mysql stop
   service mysql start
   service mysql restart
   service mysql status
